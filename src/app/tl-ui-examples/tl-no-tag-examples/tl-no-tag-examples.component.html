tl-no-tag examples

There's no such thing as a 'tl-no-tag' component -- we will not remove the wrapping tag here.

This is just a hack -- you can bring the template of a component out of that component.

The result will be that the componet tag and its template standing next to each other, instead of component tag wrapping its template.

It's not straight forward. Please checkout the source code of this example for detail.
